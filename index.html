    Виджеты
    Из виджетов собираются шаблоны. Виджеты состоят из следующих файлов:
    • Сниппет snippet.liquid
    • SCSS snippet.scss
    • Переводы messages.json
    • Форма настроек settings_form.json
    • Настройки по умолчанию settings_data.json
    • Блоки по умолчанию setup.json
    • Превью preview.jpg и mobile_preview.jpg
    • Метаданные info.json
    Шаблоны
    Шаблоны состоят из виджетов и файлов с настройками
    • Setupsetup.json
    • Settings settings.json
    Core.css
    Core.css являются стилями по умолчанию, которые можно использовать, чтобы облегчить написание кода в snippet.liquid и в snipet.scss
    Иконки
    Есть иконки редактора которые можно прописывать в настройках  settings_form.json
    Есть иконки шаблонов которые мы можем использовать в  snippet.liquid используя классы.
    Технологии
    Liquidhub
    Теги управляют логикой шаблонов, используются в Сниппет snippet.liquid
    SCSS
    Используется в SCSS snippet.scss
    Javascript
    Плагины и Фреймворки
    Подключаются в файле info.json, а настройки прописываются в файле snipet.js
    • splide@3.6.9
    • commonjs_v2
    • my-layout
    • microalert
    • vanilla-lazyload
    • fslightbox
    • micromodal
    • body-scroll-lock
    • js-cookie
    • cut-list
    • nouislider
    JSON
    Формат json используется в messages.json, settings_form.json, settings_data.json, setup.json и info.json

    Используемые понятия

    Редактор сайта - Сайдбар слева сайта клиента с настройками шаблона и виджетов
    Панель администратора - В панели администратора можно загрузить шаблоны и виджеты и редактировать код
